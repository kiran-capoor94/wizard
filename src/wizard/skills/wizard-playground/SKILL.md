# Wizard Playground — Mermaid Diagram Workbench

## What You Produce

A single self-contained HTML file, written to `<descriptive-name>.html`, then opened with `open <filename>.html`.

The file contains:
- A **left panel** (40% width) with an editable textarea pre-loaded with the Mermaid source
- A **right panel** (flex: 1) with the live-rendered diagram, zoom/pan enabled
- A **toolbar** with a preset selector, Copy Source, Copy SVG, zoom +/-, Fit, and 1:1 buttons
- Live re-render on every keystroke (debounced 300ms)
- Dark theme throughout

## Critical Rendering Rules

These are lessons learned from debugging — follow exactly or diagrams will be blank, illegible, or broken:

1. **`useMaxWidth: false`** — MUST be set for every diagram type in `mermaid.initialize()`. Without it Mermaid forces `width="100%"` on the SVG which stretches it to browser width and appears blurry/tiny.
2. **`mermaid.render(id, src)`** — use this, NOT `mermaid.run()`. Returns `{ svg, bindFunctions }`. The `svg` string must be injected via `container.innerHTML = svg` — this is the canonical Mermaid pattern per official docs. The SVG is machine-generated by Mermaid, not user HTML, so this is safe.
3. **`bindFunctions?.(container)`** — call after injecting so interactive elements wire up.
4. **Pan-layer layout** — `#preview` must be `position: relative; overflow: hidden`. `#pan-layer` must be `position: absolute; top: 0; left: 0; transform-origin: 0 0`. This is the only layout that works for zoom/pan without conflicting with Mermaid's SVG dimensions.
5. **`requestAnimationFrame(fitDiagram)`** after render — wait one frame so `viewBox` is available before computing fit scale.
6. **Sequence diagram labels** — no parentheses + commas e.g. avoid `foo(a, b)`. Write `foo: a + b` instead. Mermaid's parser misreads `(` as a method call token.

## Full HTML Template

Copy this exactly. Replace only the PRESETS content and the final `loadPreset()` call.

```html
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Diagram Playground</title>
<script src="https://cdn.jsdelivr.net/npm/mermaid/dist/mermaid.min.js"></script>
<style>
  * { box-sizing: border-box; margin: 0; padding: 0; }
  body { display: flex; flex-direction: column; height: 100vh;
         background: #1a1a1a; color: #e0e0e0; font-family: monospace; }
  #toolbar { display: flex; align-items: center; gap: 8px; padding: 8px 12px;
             background: #111; border-bottom: 1px solid #333; flex-shrink: 0; }
  #toolbar select, #toolbar button {
    background: #2a2a2a; color: #e0e0e0; border: 1px solid #444;
    padding: 4px 10px; border-radius: 4px; cursor: pointer; font-size: 13px; }
  #toolbar button:hover { background: #3a3a3a; }
  #toolbar label, #toolbar span { font-size: 13px; color: #888; }
  #zoom-label { min-width: 42px; }
  #main { display: flex; flex: 1; overflow: hidden; }
  #editor { width: 40%; border-right: 1px solid #333; display: flex; flex-direction: column; flex-shrink: 0; }
  #source { flex: 1; background: #111; color: #ccc; border: none; outline: none;
            padding: 12px; font-family: monospace; font-size: 12px; resize: none; }
  #error { color: #f66; font-size: 12px; padding: 4px 12px; background: #1a1a1a;
           border-top: 1px solid #333; min-height: 22px; flex-shrink: 0; }
  #preview { flex: 1; overflow: hidden; position: relative;
             background: #1a1a1a; cursor: grab; user-select: none; }
  #preview.dragging { cursor: grabbing; }
  #pan-layer { position: absolute; top: 0; left: 0;
               transform-origin: 0 0; padding: 24px; white-space: nowrap; }
  #pan-layer svg { display: block; }
</style>
</head>
<body>
<div id="toolbar">
  <label for="preset">Preset:</label>
  <select id="preset" onchange="loadPreset(this.value)">
    <option value="flowchart">Flowchart</option>
    <option value="sequence">Sequence</option>
    <option value="erd">ERD</option>
    <option value="state">State Machine</option>
  </select>
  <button id="btnCopySource" onclick="copySource()">Copy Source</button>
  <button id="btnCopySVG" onclick="copySVG()">Copy SVG</button>
  <button onclick="zoomStep(1)">+</button>
  <button onclick="zoomStep(-1)">-</button>
  <button onclick="fitDiagram()">Fit</button>
  <button onclick="resetView()">1:1</button>
  <span id="zoom-label">100%</span>
</div>
<div id="main">
  <div id="editor">
    <textarea id="source" oninput="scheduleRender()" spellcheck="false"></textarea>
    <div id="error"></div>
  </div>
  <div id="preview">
    <div id="pan-layer"></div>
  </div>
</div>
<script>
mermaid.initialize({ startOnLoad: false, theme: 'dark',
  flowchart: { useMaxWidth: false },
  sequence:  { useMaxWidth: false },
  er:        { useMaxWidth: false },
  state:     { useMaxWidth: false },
});

const preview  = document.getElementById('preview');
const panLayer = document.getElementById('pan-layer');
const zoomLbl  = document.getElementById('zoom-label');
let scale = 1, tx = 24, ty = 24;
let dragging = false, ox = 0, oy = 0;

function applyTransform() {
  panLayer.style.transform = 'translate(' + tx + 'px,' + ty + 'px) scale(' + scale + ')';
  zoomLbl.textContent = Math.round(scale * 100) + '%';
}
function zoomStep(dir) {
  const s = Math.min(4, Math.max(0.05, scale * (dir > 0 ? 1.25 : 0.8)));
  const cx = preview.clientWidth / 2, cy = preview.clientHeight / 2;
  tx = cx - (cx - tx) * (s / scale);
  ty = cy - (cy - ty) * (s / scale);
  scale = s; applyTransform();
}
function fitDiagram() {
  const svg = panLayer.querySelector('svg');
  if (!svg) return;
  const vb = svg.viewBox && svg.viewBox.baseVal;
  const w = (vb && vb.width)  ? vb.width  : svg.scrollWidth;
  const h = (vb && vb.height) ? vb.height : svg.scrollHeight;
  if (!w || !h) return;
  scale = Math.min(1, (preview.clientWidth - 48) / w, (preview.clientHeight - 48) / h);
  tx = 24; ty = 24; applyTransform();
}
function resetView() { scale = 1; tx = 24; ty = 24; applyTransform(); }

preview.addEventListener('wheel', function(e) {
  e.preventDefault();
  const r = preview.getBoundingClientRect();
  const cx = e.clientX - r.left, cy = e.clientY - r.top;
  const s = Math.min(4, Math.max(0.05, scale * (e.deltaY < 0 ? 1.12 : 0.9)));
  tx = cx - (cx - tx) * (s / scale);
  ty = cy - (cy - ty) * (s / scale);
  scale = s; applyTransform();
}, { passive: false });
preview.addEventListener('mousedown', function(e) {
  if (e.button !== 0) return;
  dragging = true; ox = e.clientX - tx; oy = e.clientY - ty;
  preview.classList.add('dragging');
});
window.addEventListener('mousemove', function(e) {
  if (!dragging) return;
  tx = e.clientX - ox; ty = e.clientY - oy; applyTransform();
});
window.addEventListener('mouseup', function() {
  dragging = false; preview.classList.remove('dragging');
});

let debounceTimer, seq = 0;
function scheduleRender() {
  clearTimeout(debounceTimer);
  debounceTimer = setTimeout(doRender, 300);
}
async function doRender() {
  const src = document.getElementById('source').value.trim();
  const errEl = document.getElementById('error');
  errEl.textContent = '';
  panLayer.textContent = '';
  if (!src) return;
  const mySeq = ++seq;
  try {
    const { svg, bindFunctions } = await mermaid.render('mmd' + mySeq, src);
    if (mySeq !== seq) return;
    panLayer.textContent = '';
    panLayer.innerHTML = svg;
    if (bindFunctions) bindFunctions(panLayer);
    requestAnimationFrame(fitDiagram);
  } catch (e) {
    if (mySeq !== seq) return;
    panLayer.textContent = '';
    errEl.textContent = String(e.message || e).split('\n')[0];
  }
}

// ── REPLACE these preset bodies with the user's diagram content ───────────────
const PRESETS = {
  flowchart: `flowchart TD
    A[Start] --> B{Decision}
    B -- Yes --> C[Action A]
    B -- No --> D[Action B]
    C --> E[End]
    D --> E`,
  sequence: `sequenceDiagram
    participant Client
    participant API
    participant DB
    Client->>API: POST /request
    API->>DB: query
    DB-->>API: rows
    API-->>Client: 200 OK`,
  erd: `erDiagram
    USER ||--o{ ORDER : places
    ORDER ||--|{ LINE_ITEM : contains
    PRODUCT }|--|{ LINE_ITEM : included_in`,
  state: `stateDiagram-v2
    [*] --> Idle
    Idle --> Processing : start
    Processing --> Done : complete
    Processing --> Error : fail
    Error --> Idle : reset
    Done --> [*]`
};

function loadPreset(key) {
  document.getElementById('source').value = PRESETS[key] || '';
  scheduleRender();
}
function copySource() {
  navigator.clipboard.writeText(document.getElementById('source').value)
    .then(function() { flashButton('btnCopySource', 'Copied!'); }).catch(function() {});
}
function copySVG() {
  const svg = panLayer.querySelector('svg');
  if (!svg) return;
  navigator.clipboard.writeText(svg.outerHTML)
    .then(function() { flashButton('btnCopySVG', 'Copied!'); }).catch(function() {});
}
function flashButton(id, label) {
  const btn = document.getElementById(id);
  const orig = btn.textContent;
  btn.textContent = label;
  setTimeout(function() { btn.textContent = orig; }, 1200);
}

loadPreset('flowchart');
</script>
</body>
</html>
```

## Workflow

1. **Understand** what the user wants to diagram — ask one clarifying question if the scope is ambiguous.
2. **Choose the closest preset** as the starting point (flowchart, sequence, ERD, state machine).
3. **Write the Mermaid source** for the user's diagram. Replace the PRESETS entry body with the actual diagram. Call `loadPreset('<key>')` at the end so the right tab loads.
4. **Write the HTML file** with a descriptive filename, e.g. `api-layers-diagram.html`.
5. **Run `open <filename>.html`** to open it in the browser.
6. Tell the user: scroll to zoom, drag to pan, Fit scales to window, 1:1 restores full size.

## What NOT to Do

- Do NOT use `mermaid.run()` — always use `mermaid.render(id, src)` with `innerHTML = svg`
- Do NOT omit `useMaxWidth: false` — causes illegible full-width SVGs
- Do NOT use `DOMParser` to inject the SVG — `innerHTML` is the correct and canonical approach
- Do NOT put `overflow: auto` on `#preview` — use `overflow: hidden` with an absolute `#pan-layer`
- Do NOT generate raw SVG `<path>` elements — Mermaid handles all SVG generation
- Do NOT output Mermaid source as a fenced code block only — always produce the HTML file
- Do NOT suggest mermaid.live or any external tool — the file is self-contained
- Do NOT use parentheses + commas in sequence diagram message labels
