# Wizard v1.1.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the local-first MCP memory layer that syncs Jira/Krisp/Notion, scrubs PII at every boundary, persists structured notes and summaries, and surfaces compounding context across sessions.

**Architecture:** SQLite (via SQLModel) is the durable store. FastMCP exposes 7 tools as the agent interface. A thin service layer (SyncService, WriteBackService, SecurityService) mediates between external integrations and the DB. All content crosses the security boundary through `SecurityService.scrub()`.

**Tech Stack:** Python 3.12, FastMCP v3, SQLModel, SQLite, Alembic, httpx, pydantic-settings
